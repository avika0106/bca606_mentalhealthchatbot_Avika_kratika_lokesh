📌 Overview
The AI-Based Mental Health Chatbot is an intelligent conversational system designed to provide emotional support and basic mental health assistance to users. The chatbot interacts with users in a natural and empathetic manner using Natural Language Processing (NLP) techniques.
This project aims to make mental health support more accessible, especially for individuals who are hesitant to seek professional help due to stigma, cost, or lack of availability.

❗ Problem Statement
Mental health issues such as stress, anxiety, and depression are increasing rapidly, yet access to professional mental health services remains limited. Many individuals do not seek help due to social stigma or lack of awareness.

There is a need for a system that provides:
Immediate support
Anonymous interaction
24/7 availability
This project attempts to bridge that gap using AI-driven conversation.

🎯 Objectives
Provide real-time emotional support through a chatbot
Analyze user input using NLP techniques
Detect emotions and sentiment in user messages
Suggest basic coping strategies and responses
Ensure privacy and anonymity of users

✨ Features
💬 Interactive chatbot interface
😊 Sentiment analysis of user input
🧠 Emotion-aware responses
🔒 Privacy-focused design
⚡ Real-time response generation

🛠️ Tech Stack
Programming Language: Python
Libraries/Frameworks: NLTK / spaCy / Transformers
Machine Learning: NLP-based models
Backend: Flask / Django
Frontend: HTML, CSS, JavaScript

🏗️ System Architecture
User Input → Text Processing → Sentiment Analysis → Response Generation → Chatbot Reply

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/avika0106/bca606_mentalhealthchatbot_Avika_kratika_lokesh.git
Navigate to the project directory:
cd medical_ChatBot
Install required dependencies:
pip install -r requirements.txt
Run the application:
python app.py

▶️ Usage
After running the application, you will have to copy the local host address that it shows and copy paste it on google or any other search engine and then the project will start.

⚠️ Limitations
The chatbot is not a replacement for professional therapy
Limited understanding of complex emotional conditions
Accuracy depends on training data and model performance

⚠️ Disclaimer
This chatbot is intended for educational and supportive purposes only. It is not a substitute for professional mental health care. If you are experiencing severe emotional distress, please seek help from a licensed mental health professional.
